Тестирование API.

При тестировании через API я проверял получение списка пользователей, получение конкретного пользователя, серверную фильтрацию приложения, создание клиента, редактирование клиента, удаление клиента .В процессе тистирования мною было найдено и зафиксировано в API 7 багов и баг-репорте 6 багов.
